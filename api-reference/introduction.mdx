## Welcome

This documentation is for developers at OTT aggregators who want to aggregate content from an AscendOTT client OTT platform. 

## Authentication

All API endpoints are authenticated using x-api-key in the HTTP header.

```json
"security": [
    { "ApiKeyAuth": [] }
  ]
```
